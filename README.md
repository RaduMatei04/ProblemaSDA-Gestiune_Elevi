# ProblemaSDA-Gestiune_Elevi
Aplicația gestionează elevii unei școli: se citește numărul lor, apoi pentru fiecare elev numele, codul de identificare și notele la 9 materii. În funcție de o comandă (1–4), programul afișează datele elevilor, elevul cu media cea mai mare, elevii cu media sub 6 sau mediana notelor unui elev identificat prin cod.
